#### Вариант 18: Система информации об автобусных рейсах и билетах

Система управления информацией об автобусных рейсах, наличии билетов и ценах на них, а также продажи билетов

##### Поддерживаемые данные
###### Рейсы:
- Компания, номер
- Пункты убытия и прибытия, промежуточные остановки
- Время и даты всех остановок
- Стоимость билетов для всех пар остановок
- Количество мест и наличие свободных мест с учетом промежуточных остановок
###### Клиенты:
- ФИО
- Контактная информация: адрес, телефон, e-mail
- Заказанные билеты

##### Поддерживаемые операции
- Получение списка рейсов по датам, направлениям и промежуточным остановкам, информации о ценах билетов и наличии свободных мест
- Получение списка клиентов, в т.ч. ехавших определенным рейсом, любыми рейсами компании, заказавших билеты
- Получение истории заказов клиента
- Заказ билетов на выбранный рейс между выбранными пунктами
- Добавление и удаление рейса, чтение и редактирование данных о нем
- Добавление и удаление клиента, чтение и редактирование данных о нем



### Сценарии использования
1. Бронирование билета на рейс
    - Перейти на главную страницу
    - Использовать поиск рейсов или перейти в список всех рейсов
    - Выбрать подходящий рейс, нажать "Забронировать"
    - На странице бронирования ввести данные
    - Подтвердить бронирование

2. Добавление нового рейса
    - Перейти на главную страницу
    - Перейти на страницу списка рейсов
    - Нажать "Добавить новый рейс"
    - Заполнить все необходимые поля формы (номер, компания, маршрут, цена, места)
    - Подтвердить добавление рейса

3. Связь с клиентами
    - Перейти на главную страницу
    - Перейти на страницу списка клиентов
    - Найти нужного клиента
    - Перейти на персональную страницу клиента
    - Просмотреть историю бронирований и контактную информацию

4. Регистрация нового клиента
    - Перейти на главную страницу
    - Перейти на страницу регистрации
    - Заполнить персональные данные и контактную информацию
    - Подтвердить регистрацию

5. Поиск рейсов по направлению
    - Перейти на главную страницу
    - Заполнить форму поиска: указать первую остановку и/или конечную остановку
    - Система покажет подходящие рейсы
    - Выбрать нужный рейс для дальнейшего бронирования

6. Управление данными клиента
    - Найти клиента через список клиентов
    - Перейти на его персональную страницу
    - Редактировать контактную информацию
    - Сохранить изменения


### Описание интерфейса
1. Главная страница
    - Форма поиска рейсов
    - Ссылка на список всех рейсов
    - Ссылка на страницу регистрации
    - Ссылка на список всех клиентов

2. Страница со списком рейсов
    - Таблица всех рейсов с информацией
    - Кнопка "Забронировать" для каждого рейса с доступными местами
    - Ссылка "Добавить новый рейс"
    - Сортировка по цене, остановках

3. Страница со списком клиентов
    - Список всех клиентов с основной информацией: ФИО, email, бонусные мили
    - Кликабельные ссылки на персональные страницы клиентов
    - Поиск клиента по имени или email

4. Страница регистрации
    - Поле login
    - Поле password
    - Ссылка "вернуться на страницу"

5. Персональная страница клиента
    - Полная информация о клиенте: ФИО, контактные данные, адрес
    - История бронирований клиента с указанием статуса (забронирован/оплачен/отменен)
    - Возможность редактирования данных клиента

6. Страница компании
    - Информация о компании: рейсы, цены
    - Возможность редактирования

7. Страница бронирования билета
    - Информация о выбранном рейсе
    - Информация о клиенте
    - Кнопка подтверждения бронирования

8. Страница добавления рейса
    - Форма для ввода данных рейса: номер, информация об остановках, общее количество мест
    - Валидация введенных данных
    - Кнопка сохранения рейса
